# Frontend 로컬 실행 가이드

## 🚀 빠른 시작

### 1. Frontend 디렉토리로 이동
```bash
cd /root/Frontend
```

### 2. 의존성 설치 (처음 한 번만)
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

### 4. 브라우저에서 접속
- 자동으로 브라우저가 열립니다
- 또는 수동으로 접속: `http://localhost:5173`
- 다른 기기에서 접속: `http://[서버IP]:5173`

---

## ✅ 수정된 내용

### 1. 로컬 실행 설정
- `vite.config.js`에 `server` 설정 추가
- `host: '0.0.0.0'` - 모든 네트워크 인터페이스에서 접근 가능
- `port: 5173` - 기본 포트
- `open: true` - 브라우저 자동 열기

### 2. 회원가입 기능
- ✅ DB 저장 없이 환영 인사만 표시
- ✅ 회원가입 후 자동으로 로그인 화면으로 전환
- ✅ 입력한 이름으로 환영 메시지 표시

### 3. 로그인 기능
- ✅ 인증 절차 없이 바로 로그인 처리
- ✅ 더미 토큰 생성하여 로그인된 상태로 처리
- ✅ 이메일/비밀번호만 입력하면 바로 로그인

---

## 📋 테스트 방법

### 회원가입 테스트
1. 브라우저에서 `http://localhost:5173` 접속
2. "지금 가입하세요." 클릭
3. 이름, 성, 이메일, 비밀번호 입력
4. "지금 가입하기" 버튼 클릭
5. ✅ 환영 인사 표시 후 로그인 화면으로 전환 확인

### 로그인 테스트
1. 로그인 화면에서 이메일, 비밀번호 입력 (아무거나 가능)
2. "로그인" 버튼 클릭
3. ✅ 인증 없이 바로 로그인 처리 확인

## 🔧 문제 해결

### 포트가 이미 사용 중인 경우
```bash
# 다른 포트 사용
npm run dev -- --port 3000
```

### 의존성 오류
```bash
# node_modules 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install
```

### 브라우저가 자동으로 안 열리는 경우
- 수동으로 `http://localhost:5173` 접속

---

## 📝 참고사항

- Backend 연결 없이 Frontend만으로 동작합니다
- 회원가입/로그인은 모두 프론트엔드에서 처리됩니다
- 실제 DB 저장이나 인증은 없습니다 (발표용 데모)

---
